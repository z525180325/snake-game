# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project

Web-based Snake game (贪吃蛇) — a beginner learning project built with vibe coding.

## Tech Stack

- HTML5 Canvas for game rendering
- CSS for UI styling
- Vanilla JavaScript (no framework, no build tools)
- Zero dependencies

## How to Run

Open `index.html` in a browser. No server or build step required.

## Architecture (Planned)

Single-page app with three layers:
- **index.html** — page structure, Canvas element, scoreboard
- **style.css** — visual design, layout, animations
- **game.js** — game loop, snake movement, collision detection, food spawning, scoring

## User Context

The user is a Chinese-speaking beginner learning software development through this project. Explain concepts clearly but concisely. Write code in English (variable names, comments can be bilingual). This is a "vibe coding" project — the AI writes the code, the user learns by reading, running, and asking questions.
